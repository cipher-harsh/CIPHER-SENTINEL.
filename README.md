# 🛡️ CIPHER-SENTINEL

**CIPHER-SENTINEL** is a Python-based cybersecurity tool for real-time system monitoring and threat detection.

## 🚀 Key Features
* **Heuristic Process Scan:** Identifies potential malware signatures.
* **Live System Stats:** Monitors CPU, RAM, and Network IP status.
* **Encrypted Secure Logs:** Automatically generates encrypted `.dat` logs on the Desktop.
* **Admin-Only Access:** High-level encryption prevents unauthorized log reading.

## 📂 Download & Use
1. Download the `CLIENT VERSION.exe` from the repository.
2. Run the application (No installation required).
3. Check your Desktop for the `sentinel_core.dat` file after a scan.

## 🔐 Need to read the Logs?
The log file (`sentinel_core.dat`) is encrypted for security. 

> [!CAUTION]
> **Agar aapko logs ko readable `.txt` format mein dekhna hai, toh Master Admin Tool aur Decryption Key ke liye mujhse contact karein.**
